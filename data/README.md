# Data

This folder stores project datasets.

```text
data/processed/uci_ckd.csv       CKD / No CKD model dataset
data/processed/nhanes_ckd.csv    CKD stage screening dataset
data/processed/data_dictionary.md
data/raw/food_data_csv/          USDA food data for nutrition features
```

Raw CKD source downloads from UCI and NHANES are reproducible through `scripts/extract_datasets.py` and are ignored by Git.
